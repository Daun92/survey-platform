## API 엔드포인트 (추가분)

| Method | Path | Auth | 설명 |
|--------|------|------|------|
| GET | /api/v1/projects | JWT | 프로젝트 목록 |
| POST | /api/v1/projects | JWT | 프로젝트 생성 |
| GET | /api/v1/projects/:id | JWT | 프로젝트 상세 |
| PUT | /api/v1/projects/:id | JWT | 프로젝트 수정 (소유자/ADMIN) |
| DELETE | /api/v1/projects/:id | JWT | 프로젝트 삭제 (소유자/ADMIN) |
| GET | /api/v1/surveys?projectId= | JWT | 설문 목록 (프로젝트별) |
| POST | /api/v1/surveys | JWT | 설문 생성 |
| GET | /api/v1/surveys/:id | JWT | 설문 상세 |
| PUT | /api/v1/surveys/:id | JWT | 설문 수정 |
| PATCH | /api/v1/surveys/:id/status | JWT | 설문 상태 변경 |
| DELETE | /api/v1/surveys/:id | JWT | 설문 삭제 |

## Phase 1 완료 요약

Phase 1 전체 완료. 구축된 기능:
