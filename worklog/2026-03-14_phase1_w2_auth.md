# Phase 1 W2: Auth 모듈, User/Department 엔티티, 로그인 페이지

- **날짜**: 2026-03-14
- **Phase**: 1 (W2)
- **작업자**: agent-infra-1

## 완료 항목

### Shared 타입 추가
- [x] `UserRole` enum (admin, manager, user)
- [x] `UserResponse`, `AuthTokenResponse`, `DepartmentResponse` 인터페이스
- [x] `LoginRequest`, `RegisterRequest` 인터페이스

### Backend 엔티티 (2개)
- [x] `Department` — UUID PK, name, code(unique), description
- [x] `User` — UUID PK, email(unique), password(bcrypt), name, role(enum), department 관계

### Backend Auth 모듈 (10개 파일)
- [x] `AuthModule` — JWT + Passport, LocalStrategy, JwtStrategy
- [x] `AuthController` — POST /auth/login, POST /auth/register, GET /auth/me
- [x] `AuthService` — validateUser, login(JWT 발급), register, getMe
- [x] `LoginDto`, `RegisterDto` — class-validator 검증
- [x] `LocalAuthGuard`, `JwtAuthGuard`
- [x] `JwtStrategy`, `LocalStrategy`

### Backend User/Department 모듈
- [x] `UsersModule` — findByEmail, findById, create, updateLastLogin
- [x] `DepartmentsModule` — findAll, findById, create
- [x] `DepartmentsController` — GET /departments, GET /departments/:id, POST /departments

### Frontend (2개 파일)
- [x] `/login` 페이지 — 이메일/비밀번호 폼, 에러 처리, 로딩 상태
- [x] `lib/api.ts` — fetch 래퍼, JWT 토큰 자동 첨부

## API 엔드포인트

| Method | Path | Auth | 설명 |
|--------|------|------|------|
| POST | /api/v1/auth/login | - | 로그인 (email, password → JWT) |
| POST | /api/v1/auth/register | - | 회원가입 |
| GET | /api/v1/auth/me | JWT | 현재 사용자 정보 |
| GET | /api/v1/departments | - | 부서 목록 |
| GET | /api/v1/departments/:id | - | 부서 상세 |
| POST | /api/v1/departments | JWT | 부서 생성 |
